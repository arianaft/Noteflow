# Gestión del proyecto

## Metodología

Tablero Kanban en Trello con cinco columnas: Backlog, Todo, In Progress,
Review y Done. Cada funcionalidad principal es una tarjeta con subtareas
técnicas. Las tarjetas se mueven a medida que avanza el desarrollo.

## Criterio para mover tarjetas

- **Todo**: priorizado para la sesión actual
- **In Progress**: hay código escrito para esa funcionalidad
- **Review**: funciona en el simulador, pendiente de revisar
- **Done**: probado y commiteado en main
